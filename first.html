How to push in github

  1. Open your terminal
  2. Navigate to your project directory
  3. First initialize a git repository by running git init
  4. You can add a specific file by running git add <filename>
  5. Run the command git add . to stage your changes
  6. Run the command git commit -m "your commit message" to commit your changes
  7. Run the command git push origin main to push your changes to the main branch
