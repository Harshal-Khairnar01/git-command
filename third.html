Git push -u Example

The -u flag sets the upstream branch so you can push and pull without specifying branch name every time.

Without -u

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/you/repo.git
git push origin main

# Future pushes
git push origin main
git pull origin main

With -u

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/you/repo.git
git push -u origin main   # sets upstream tracking

# Future pushes (short form)
git push
git pull

Note: You can use -u with any branch, not just main.
