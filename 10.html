Understanding git commit --amend

The git commit --amend command allows you to modify the most recent commit. You can use it to :

  • Change the commit message.
  • Add or remove files from the last commit.
  • Combine changes without creating a new commit.

git add file.txt
git commit --amend -m "Updated commit with new file"
