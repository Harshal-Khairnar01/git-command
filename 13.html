Deploying a Static Website to GitHub Pages

What You Need

  • A GitHub account (Sign up here)
  • Git installed on your computer (Download Git)
  • Your static website files (HTML, CSS, JS)

Steps to Deploy

  1. Create a Repository
    • Go to GitHub and click New repository.
    • Give it a name (e.g., my-website) and make it public.
  2. Initialize Git and Push Your Code 
    cd path/to/your/project
git init
git remote add origin https://github.com/your-username/my-website.git
git add .
git commit -m "Initial commit"
git push -u origin main
  3. Enable GitHub Pages
    • Go to your repositoryâ€™s Settings â†’ Pages.
    • Under Source, select main branch and /root folder.
    • Click Save.
  4. Access Your Website
    • After a few minutes, your website will be live at:
      https://your-username.github.io/my-website/

Tips

  • Keep your project in the main branch for simpler deployment.
  • You can update the site by committing and pushing new changes.
  • For custom domains, configure them under the Pages settings.
